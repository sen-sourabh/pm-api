## Description

Vault - The Password Manager that allow to keep your password organized, encrypted, and more accessible.

## Running the app

```bash
# local - Please take a fork first
$ git clone <your_git_url>/pm-api.git
$ npm run init:vault

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
